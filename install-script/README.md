# راهنمای اجرای اسکریپت نصب Wyron

این فایل توضیح می‌دهد چگونه اسکریپت نصب Wyron را اجرا کنید.

---

## 1️⃣ قرار دادن فایل‌ها

فایل‌های زیر را داخل یک پوشه قرار دهید:

```
wyron          (فایل باینری پنل)
install.sh     (اسکریپت نصب)
README.md
```

---

## 2️⃣ دادن دسترسی اجرا به اسکریپت

داخل همان پوشه دستور زیر را اجرا کنید:

```bash
chmod +x install.sh
```

---

## 3️⃣ اجرای اسکریپت

اگر کاربر عادی هستید:

```bash
sudo ./install.sh
```

اگر با کاربر root هستید:

```bash
./install.sh
```

---

## 4️⃣ مراحل نصب

در هنگام اجرا، اسکریپت از شما موارد زیر را می‌پرسد:

- مسیر فایل باینری Wyron
- نصب MongoDB (در صورت نصب نبودن)
- نام کاربری و رمز MongoDB
- پورت پنل
- آدرس Listen
- دامنه اصلی (اختیاری)
- تنظیمات TLS

مقادیر را مطابق نیاز خود وارد کنید.

---

## 5️⃣ فعال‌سازی لایسنس

در اولین اجرا، پنل یک Device Fingerprint نمایش می‌دهد.

آن را برای پشتیبانی ارسال کنید تا فایل `license.key` دریافت کنید.

فایل لایسنس را در مسیر زیر قرار دهید:

```
/opt/wyron/license.key
```

سپس پنل را اجرا کنید:

```bash
/opt/wyron/wyron start
```

---

## 6️⃣ بررسی وضعیت سرویس

برای مشاهده وضعیت پنل:

```bash
systemctl status wyron
```

برای مشاهده لاگ‌ها:

```bash
journalctl -u wyron --no-pager
```

---

پوشه نصب پیش‌فرض:

```
/opt/wyron
```

فایل تنظیمات:

```
/opt/wyron/config.yml
```
